Arte generativo 🎨

Microalgas verdes apartir de código

Como ilustradora científica, microbióloga y desarrolladora web, he comenzado a explorar el arte generativo: una disciplina visual que combina código y creatividad para producir composiciones automatizadas. En esta pieza, programé una serie de microalgas verdes con núcleos internos, inspiradas en organismos reales como Chlorella y Volvox. Cada célula fue generada por código con variaciones en forma, tamaño, opacidad y posición, simulando la apariencia dinámica de una muestra observada bajo el microscopio. Este ejercicio representa una fusión entre ciencia, ilustración y desarrollo creativo, donde el código se convierte en herramienta visual.

Temática: formas microscópicas, vida acuática, simetría natural.
Técnica: Ilustración generada por código.
Programa: p5.js.
Año: 2025.
Formato: PNG.
